# Dynamic Streetlights

Dynamic Streetlights is a Minecraft mod that adds various streetlights and lamps to your world. It includes flickering effects, redstone-controlled streetlamps, and a remote lever for controlling multiple lights at once.

## Features
- **Streetlight Bottoms, Poles, and Supports**: Modular blocks to build custom streetlight structures.
- **Streetlight Heads**: Various head designs with flickering light effects.
- **Redstone Streetlight Heads**: Controlled by redstone or remote levers.
- **Remote Lever**: A specialized tool to toggle all streetlights within a 64-block radius.
